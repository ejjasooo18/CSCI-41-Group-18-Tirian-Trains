PROJECT SET UP:
1. Create first your own virtual environment (venv) and install the required dependencies.
2. Run on terminal 'python -m venv venv'
3. Activate venv 'venv\Scripts\activate'
4. Install dependencies 'pip install -r requirements.txt'
5. Don't forget to make migrations
6. Fixture: The repository comes with a fixture that you can use for seeding the database '
py manage.py loaddata dump.json'
